# Documentação

## Métodos <a id="methods"></a>

* get\(table\)
* save\(table, value\)
* remove\(table\)
* getAll\(\)



### get

```javascript
snow.get('any_bag')// undefined
```

### save

```javascript
snow.save('any_bag', 200)// 200
```

### remove

```javascript
snow.remove('any_bag')// undefined
```

### getAll

```javascript
snow.getAll()// all db's
```
