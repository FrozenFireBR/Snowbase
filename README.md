# Introdução

## Instalação



```javascript
$ npm install snowbase
```

{% hint style="warning" %}
 Você deve criar um arquivo fora de qualquer pasta chamado snowbase.json \('./snowbase.json'\)
{% endhint %}

## Requerimento

```javascript
const snow = require('snowbase')
```
